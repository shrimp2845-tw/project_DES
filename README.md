### project_DES
An educational Python implementation of the Data Encryption Standard (DES). This project breaks down the complex Feistel network into human-readable modules following the official NIST specifications, and supports ECB and CBC modes for real file encryption.

### history
The Data Encryption Standard (DES) is a symmetric block cipher developed by IBM in the 1970s.
It is was widely used in banking and secure communications for decades.
Due to its short key length, DES became vulnerable to brute-force attacks and was eventually replaced by the Advanced Encryption Standard (AES) in 2001.
Although no longer secure, DES remains historically important and is widely studied for learning modern cryptography and block cipher design.
